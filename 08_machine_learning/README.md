# Machine Learning Module

This module introduces the **core concepts and algorithms of Machine Learning using Python and Scikit-Learn**.
It builds on the previous Python, NumPy, Pandas, and Data Visualization modules and demonstrates how to **train, evaluate, and deploy machine learning models**.

The notebooks in this module cover **supervised learning, unsupervised learning, model evaluation, and complete ML pipelines**.

# Key Concepts Learned

### Machine Learning Basics

Understanding how machines learn patterns from data instead of being explicitly programmed.

### Supervised Learning

Training models using labeled datasets to make predictions.

### Unsupervised Learning

Discovering hidden patterns and groupings in unlabeled datasets.

### Feature Scaling

Normalizing and standardizing features so models perform better.

### Model Evaluation

Using metrics like accuracy, precision, recall, and F1 score to measure performance.

### Bias-Variance Tradeoff

Balancing model complexity to avoid overfitting or underfitting.

### End-to-End ML Pipeline

Complete workflow including preprocessing, training, evaluation, and prediction.
