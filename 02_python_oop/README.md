# Python Object Oriented Programming (OOP)

This folder contains notebooks covering the core concepts of Object Oriented Programming in Python.

OOP is an important programming paradigm used in large software systems and many machine learning libraries such as PyTorch and TensorFlow.

---

## Learning Order

1. **Classes and Objects**
   - Understanding classes
   - Creating objects
   - Attributes and methods

2. **Constructors**
   - `__init__` method
   - Object initialization
   - Default and parameterized constructors

3. **Inheritance**
   - Parent and child classes
   - Method overriding
   - Using `super()`

4. **Encapsulation**
   - Public, protected, and private variables
   - Getter and setter methods

5. **Polymorphism**
   - Same method behaving differently
   - Method overriding
   - Operator polymorphism

6. **Abstraction**
   - Abstract classes
   - Abstract methods
   - Using the `abc` module
