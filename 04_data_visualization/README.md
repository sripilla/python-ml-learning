# Module 04 — Data Visualization (Matplotlib & Seaborn)

This module focuses on **data visualization in Python**, an essential step in **Exploratory Data Analysis (EDA)**. Visualization helps understand patterns, trends, relationships, and anomalies in datasets before building machine learning models.

The module introduces **Matplotlib** for fundamental plotting and **Seaborn** for statistical visualizations.

---

## Learning Objectives

By completing this module, you will learn how to:

* Create basic plots using **Matplotlib**
* Customize plots with titles, labels, legends, and grids
* Use **Seaborn** for statistical visualization
* Analyze distributions and relationships between variables
* Visualize correlations between dataset features
* Perform basic **Exploratory Data Analysis (EDA)**

---

## Libraries Used

* **Matplotlib** – Core Python visualization library
* **Seaborn** – Statistical visualization built on Matplotlib
* **NumPy** – Numerical data generation
* **Pandas** – Data handling and analysis

## Why Visualization Matters in Machine Learning

Visualization helps in:

* Understanding **data distribution**
* Detecting **outliers**
* Identifying **feature relationships**
* Understanding **correlation between variables**
* Performing **EDA before model training**

Good visualization is a crucial skill for **data scientists, ML engineers, and AI practitioners**.
