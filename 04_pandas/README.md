# Pandas Module

This module covers **Pandas**, a powerful Python library used for data manipulation and analysis.

Pandas provides two main data structures:

- **Series** (one-dimensional data)
- **DataFrame** (two-dimensional tabular data)

These structures make it easy to clean, analyze, and transform datasets before applying machine learning models.
