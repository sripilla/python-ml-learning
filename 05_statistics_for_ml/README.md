# Module 05 — Statistics for Machine Learning

Statistics is a foundational component of **Machine Learning and Data Science**. Most machine learning algorithms rely on statistical principles to understand patterns in data and make predictions.

This module introduces the essential statistical concepts required for machine learning.

---

## Learning Objectives

By completing this module, you will learn:

* Descriptive statistics
* Probability fundamentals
* Data distributions
* Correlation and covariance
* Hypothesis testing basics
* Statistical intuition for machine learning

---

## Why Statistics Matters in Machine Learning

Statistics helps in:

* Understanding data behavior
* Identifying patterns and relationships
* Evaluating model performance
* Making data-driven decisions

Many ML algorithms such as **Linear Regression, Logistic Regression, and Naive Bayes** rely heavily on statistical concepts.
