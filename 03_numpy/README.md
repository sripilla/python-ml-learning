# NumPy Module

This module introduces **NumPy**, the core library used for numerical computing in Python.

NumPy provides fast multidimensional arrays and mathematical operations that are widely used in machine learning, scientific computing, and data analysis.

---

# Learning Order

1. **NumPy Basics**
   - Introduction to NumPy
   - Creating NumPy arrays
   - Array dimensions and data types

2. **Array Creation**
   - zeros()
   - ones()
   - arange()
   - linspace()
   - identity matrices
   - random arrays

3. **Indexing and Slicing**
   - Accessing array elements
   - Selecting rows and columns
   - Boolean indexing
   - Conditional selection

4. **Array Operations**
   - Element-wise operations
   - Mathematical functions
   - Aggregation functions
   - Broadcasting

5. **Reshaping and Stacking**
   - reshape()
   - flatten()
   - concatenate()
   - horizontal and vertical stacking

6. **Statistics and Linear Algebra**
   - mean, median, standard deviation
   - matrix multiplication
   - dot product
   - transpose and determinant

---
# Purpose

NumPy forms the **mathematical foundation for machine learning** and is heavily used by libraries such as:

- Pandas
- Scikit-learn
- TensorFlow
- PyTorch
